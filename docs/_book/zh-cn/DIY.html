自定制 (DIY)

§ 自定制 (DIY)

多亏有了 vue-cli 的 build 功能，让我们可以避免繁琐的 Webpack 配置

>_ git clone https://github.com/OneWayTech/vue2-datatable.git
>_ cd vue2-datatable
>_ npm i
>_ npm start

由此，您可以基于 examples/ 来定制源码 lib/
如果您觉得某些功能很必须，随时欢迎 issues / PR！
